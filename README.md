# Customer Shopping Behavior Analysis Dashboard



# Project Overview

This project analyzes customer shopping behavior using SQL Server and Power BI.
The objective is to uncover purchasing trends, customer demographics, subscription patterns, and category-wise revenue insights through interactive visualizations and SQL-based analysis.

The dashboard helps businesses understand customer behavior and make data-driven decisions.
